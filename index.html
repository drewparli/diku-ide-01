Project 1: Warming up

Since this is the first project, we'll start off slowly. The idea with this project is to gain some experience with setting up a web-page, and writing HTML, CSS and JavaScript. You will need a server somewhere to host the files that you will create in the course. We recommend that you use GitHub Pages for this purpose - this will also make sharing files among group members easier. This page https://pages.github.com/ gives a detailed walk-through of how to set it up (Note that you want to set up a Project site and that you should Start from scratch). If you have another server already set up, you are of course also welcome to use that. If github is new to you, or if you don't know which editor to use for editing HTML, Javascript and CSS files, please see Setting up a github repository and an editor.
